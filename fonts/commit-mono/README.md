# Commit Mono

Protocol Ink bundles the two upright desktop faces from Commit Mono v1.143:

- `CommitMono-400-Regular.otf`
- `CommitMono-700-Regular.otf`

Italics are intentionally omitted so terminal emphasis stays upright. The font
family exposed to applications is `CommitMono`.
